# RallyRivals master palette (ADR-003)

ONE palette shared by voxel models, terrain, FX and UI accents — coherence comes from colour
discipline, not from voxelizing the world. 12 ramps x 6 shades, darkest -> lightest.
Original palette (no external licence); generated by `scripts/tools/gen_palette.gd` — edit the
ramp endpoints there and regenerate, the PNGs are artifacts.

| file | purpose |
|------|---------|
| `rallyrivals_palette.png` | 256x1 strip — load in MagicaVoxel: Palette panel -> open -> this file |
| `rallyrivals_swatches.png` | human-readable sheet, one ramp per row |

## Ramps (slots 0..59, 6 per ramp)

| ramp | slots | use |
|------|-------|-----|
| asphalt | 0-5 | roads, dark neutrals (step 3 = SurfaceType anchor, PINNED) |
| gravel | 6-11 | gravel, light neutrals, concrete props (step 3 PINNED) |
| dirt | 12-17 | dirt roads, soil, wood (step 3 PINNED) |
| sand | 18-23 | sand/off-road, dry grass (step 4 PINNED) |
| grass | 24-29 | vegetation, tree props |
| frost | 30-35 | snow + ice, sky tints (steps 4=ice, 5=snow PINNED) |
| apex | 36-41 | Apex Cartel bodies — smoky warm graphite (retro) |
| wreck | 42-47 | Wreckhouse bodies — leather rust into burnt steel (retro) |
| mayfly | 48-53 | Mayfly Speedworks bodies — Gulf-livery powder blue (retro) |
| sulphur | 54-59 | GAME IDENTITY: key art, menus, UI (step 5 = #f5f5b8, PINNED) |
| carmine | 60-65 | GAME IDENTITY: key art, logo, highlights (step 3 = #a10b2b, PINNED) |
| accents | 66-71 | vermillion (Apex), retro orange (Wreckhouse), mustard (Mayfly), gold, sage, cream |

## Rules

- **Pinned cells never move** — they are the `SurfaceType.color` classifier anchors; painted
  track images depend on them. Re-tint anything else freely, regenerate, re-export models.
- **Never reorder existing slots** once voxel models exist — their UVs index the strip.
  New ramps append at the end only.
- Shade INSIDE a ramp (step along the row), don't cross-mix ramps for shading.
- Each brand's accent comes from the accents row, not from another brand's ramp.
- Terrain/splat colours and MagicaVoxel palettes both come from here — if a colour isn't on
  the board, it isn't in the game.
