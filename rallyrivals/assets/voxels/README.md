# Voxel models (ADR-003)

1 voxel = 0.1 m. Authored in MagicaVoxel; every model is `.vox` (source) + `.obj`/`.mtl`/`.png`
(export artifact) committed together. Stubs were generated by `scripts/tools/gen_voxel_stubs.gd`
— each `.vox` opens in MagicaVoxel with the master palette embedded and the correct envelope.

## Filling a stub

1. Open the model's `.vox` in MagicaVoxel (the block-out shell is your proportion guide).
2. Sculpt. Stay inside the envelope; keep wheel arches where they are (shared chassis:
   collision 2x1x4 m, wheels at +/-0.85 / +/-1.4). Palette colours only.
3. Export -> Wavefront (.obj), straight over the stub's `.obj`/`.png`/`.mtl` in this folder.
4. Godot reimports on focus; the car scene picks it up. Save the `.vox` too — it's the source.

## Layout

| path | what |
|------|------|
| `cars/<id>.vox|.obj` | base shell, brand-tinted block-out until sculpted |
| `cars/<id>_d1.*` / `_d2.*` | light/heavy damage variants — SCULPT AFTER code-vehicle-damage defines the read; stubs are copies of base |
| `wheels/wheel_<brand>.*` | shared wheel per brand, axle along X, radius exactly 0.4 m |

Cars: forward = +Z (MagicaVoxel +Y after the z-up mapping), envelope 18x12x40 voxels.
Wheels: 3x8x8, spin around X — no corrective rotation needed in scenes.
